# Claude Desktop CN v1.6.30

这一版是一个针对性修复版，重点收口聊天页技能菜单里的重复 Skill 问题，并把这次修复同步到 GitHub 和 Windows 安装包。

## 主要更新

### 1. 修复聊天页技能菜单重复项
- 修复 `code-review`、`create-project` 等 Skill 在聊天页加号菜单里重复显示的问题。
- 根因是内置 Skill 会同步到 `~/.claude/skills` 供运行时使用，前端又把内置列表和用户目录里的同名副本一起展示了。
- 现在后端接口会过滤这些同步副本，前端菜单也会按 Skill 名称再做一层去重。

### 2. 保持现有 Skill 绑定和触发逻辑不变
- 项目绑定、触发示例、中文说明和启用状态都保留。
- 这次只修重复展示，不改已有 Skill 的使用方式。

### 3. 发布与安装包更新
- GitHub 仓库已更新到 `v1.6.30`
- Windows 安装包已更新为 `Claude-Desktop-CN-Setup-1.6.30.exe`

## 验证

本地已通过：

```bash
npm run build
node -c electron/bridge-server.cjs
npm run electron:build:win
```
