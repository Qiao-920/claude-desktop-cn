# Claude Desktop CN v1.6.17

这一版是一次成品化收口，重点放在三个地方：协作页、设置页、代码工作区。

## 主要更新

### Cowork 协作页

- 从空白说明页升级为正式工作总览
- 增加项目、GitHub、权限、归档状态卡片
- 增加项目、代码工作区、权限设置快捷入口
- 增加当前队列和最近项目列表
- 明确 Cowork 与 Code 页的职责边界

### Settings 设置页

- 修复骨架区可编译状态
- 补齐 Git、MCP、环境、工作树、已归档聊天、使用情况页面
- 保留“骨架”标记，但每个骨架页都有状态说明和下一步入口
- 常规设置继续保留多选项结构，不再只放一个空按钮

### Code 工作区

- 增加工作流状态条
- 显示当前工作区路径
- 显示 Git 是否干净 / 是否有未提交改动
- 显示最近命令数量
- 显示当前权限模式
- 显示提交后自动推送 / 手动推送策略

## 修复与维护

- README 更新到 `1.6.17`
- 发布说明补齐本轮功能
- 保持 Windows 托盘后台运行、托盘图标、黑屏修复路线继续生效
