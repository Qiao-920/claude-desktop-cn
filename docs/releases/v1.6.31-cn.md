# Claude Desktop CN v1.6.31

这一版是一次模型与聊天体验对齐的小版本更新，重点把默认 Opus 配置升级到 4.8，并收口前几轮已经修好的聊天发送、模型识别和侧边助手交互问题。

## 主要更新

### 1. 默认 Opus 配置升级到 4.8
- 项目页默认推荐模型从 `claude-opus-4-6` 升级到 `claude-opus-4-8`
- 设置页默认模型与兜底模型列表同步切到 `claude-opus-4-8-thinking`
- 升级套餐页的 Opus 计费锚点也改成 `claude-opus-4-8`
- 研究编排默认模型从 `claude-opus-4-6` 更新为 `claude-opus-4-8`

### 2. 聊天发送链路保持使用当前所选模型
- 主聊天与代码侧栏发送消息时，会显式带上当前下拉框里选中的模型
- 不再因为旧会话模式或全局兜底，偷偷回退成 `Claude Sonnet 4.6`
- DeepSeek 自托管供应商的最新模型发现与显示链路已对齐到 `DeepSeek V4 Flash / DeepSeek V4 Pro`

### 3. 侧边助手交互修复继续保留
- 代码侧栏普通消息不再自动注入整段当前文件内容
- 右侧代码侧边助手流式输出时会自动贴底滚动到最新回复
- 聊天技能菜单中的重复 Skill 项会继续保持去重

## 发布结果

- GitHub 仓库已更新到 `v1.6.31`
- Windows 安装包已更新为 `Claude-Desktop-CN-Setup-1.6.31.exe`
