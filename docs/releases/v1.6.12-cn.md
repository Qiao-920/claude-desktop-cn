# Claude Desktop CN v1.6.12

本次版本是 `Claude Desktop CN` 分支的一次整理发布，重点不在“换版本号”，而在于把你自己的中文分支真正做成一个能持续维护和分发的版本。

## 本版重点

### 1. GitHub OAuth 切换为自有配置

- 支持使用你自己的 GitHub OAuth App
- 不再继续复用原作者的 GitHub OAuth 配置
- GitHub 导入链路可正常工作

### 2. 中文化继续推进

- 继续清理零散英文
- 增强中英文 UI 切换
- 补齐更多中文界面文案

### 3. Code 工作区增强

- 支持工作区选择
- 支持文件树浏览与编辑
- 支持 HTML 预览
- 支持新建 / 删除 / 重命名
- 支持 Git 状态与单文件差异
- 支持单文件恢复
- 支持提交 / 推送
- 新增“提交后自动推送”偏好

### 4. Settings 去骨架化

以下分区不再只是占位展示：

- Git
- MCP 服务器
- 环境
- 工作树
- 已归档聊天

### 5. Cowork 页面补强

- 从空白占位页升级为协作总览页
- 展示项目、GitHub、权限、工作区状态
- 提供跳转入口，为后续协作闭环做承接

## 安装包

Windows 安装包文件名：

```text
Claude-Desktop-CN-Setup-1.6.12.exe
```
