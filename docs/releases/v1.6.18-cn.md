# Claude Desktop CN v1.6.18

这一版重点不是新增页面，而是把发版链路补成正式可维护流程。

## 主要更新

### GitHub Actions 自动发版

- 将构建流程收敛为 Windows 优先
- 手动运行 workflow 时上传安装包 artifact
- 推送 `v*` 标签时自动创建 GitHub Release
- 自动上传：
  - `Claude-Desktop-CN-Setup-*.exe`
  - `Claude-Desktop-CN-Setup-*.exe.blockmap`
  - `latest.yml`

### CI 稳定性

- 移除 macOS / Linux 矩阵构建，避免非当前目标平台拖垮发布
- 使用 `npm ci` 保证依赖安装可复现
- 使用 `actions/setup-node` 的 npm 缓存加速构建
- 在 Windows runner 上复制 Bun 运行时到 `engine/bin/bun.exe`
- 关闭自动证书发现，避免 GitHub runner 因没有本地签名证书而误判

### OAuth 配置

- 构建时从 GitHub Actions secrets 写入 `electron/secrets.json`
- 支持：
  - `GH_OAUTH_CLIENT_ID`
  - `GH_OAUTH_CLIENT_SECRET`
- 如果没有配置 secrets，应用仍能正常启动，只是 GitHub 导入功能会提示缺少 OAuth 配置。

## 安装包

Windows 安装包：

```text
Claude-Desktop-CN-Setup-1.6.18.exe
```
