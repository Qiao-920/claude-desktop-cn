# Claude Desktop CN v1.6.14

`1.6.14` 是一个小版本修复版，重点收口 Windows 托盘图标问题。

## 本版重点

### 1. 修复 Windows 托盘图标空白

之前在部分 Windows 环境里，应用关闭窗口进入后台后：

- 托盘位置已经存在
- 进程也还在运行
- 但图标会显示成一块空白

这一版改成了更稳的加载方式：

- Windows 下优先直接使用 `.ico` 托盘图标
- 补了高 DPI 缩放兜底
- 路径异常时再走回退逻辑

### 2. 后台驻留逻辑保持不变

这次没有回退后台驻留功能。

也就是说现在的行为是：

- 点击关闭按钮
- 主窗口隐藏
- 应用继续驻留后台
- 可以通过系统托盘重新打开或退出

### 3. 版本线继续往前走

从这版开始，继续按增量迭代往后走，不再停留在 `1.6.12` 的命名上。

## 安装包

```text
Claude-Desktop-CN-Setup-1.6.14.exe
```
