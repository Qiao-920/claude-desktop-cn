# Claude Desktop CN v1.6.13

`1.6.13` 是这一轮工程化收口版本。

这次重点不是继续铺更多新页面，而是把已经接上的能力补顺，尤其是后台驻留、技能导入、模式切换隔离和发布链。

## 本版重点

### 1. 关闭窗口后转入托盘驻留

在 Windows 上关闭主窗口后，应用不再直接退出，而是：

- 隐藏主窗口
- 保持后台运行
- 在系统托盘保留入口
- 可以通过托盘菜单重新显示或彻底退出

这让桌面客户端更接近日常常驻工具的行为。

### 2. 支持导入 Skill

客户端现在支持直接导入 Skill：

- `.zip`
- `.md`

导入后会写入本地 Skill 目录，后续可以在客户端里继续使用。

### 3. 修复模式切换时的配置串味

切换自部署 / Clawparrot 模式时：

- 会清理不该继续复用的模型配置
- 会避免自部署模型残留到 Clawparrot 模式
- 会减少模式切换后出现“模型列表对不上”的情况

### 4. 清理发布工作流

移除了原作者工作流里依赖外部私有服务器密钥的同步步骤。

现在你自己的仓库保留的是：

- 正常构建
- 正常打包
- 正常发布

不会再因为作者私有环境缺失而导致 CI 红掉。
